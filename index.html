About Smart Goals

A smart goals is widely known framework that helps turn ambigous intentions into actionable plans which can betracked and measured.

It is usually considered to a five word mnemonic acronym to help define objectives in precise details.


Features of a smart goal

  1. Helps with the creationg and management of goals.
  2. Helps to monitor and track progress thus keeping one motivated.
  3. It help set reminds that can be used as oacesetters to keep on track.
  4. It helps the community to connect and share their journey together.

Our Smart Goals

  1. By the end of five weeks we want to be able to learn basic structures of programming languages JavaScript, CSS and HTML.
  2. In a span of five weeks we want to be able to build a simple web application using the skills we have learned.
  3. By the end of the course we want to have completed a portfolio of projects that showcase our abilities.
